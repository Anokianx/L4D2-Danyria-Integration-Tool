# L4D2-Danyria-Integration-Tool
Danyria is an open-source toolkit for Left 4 Dead 2, designed to provide mod management, external HUD overlays, weapon data editing, and local plugin integration in one desktop application.  

Danyria 是一个面向《求生之路2》的开源整合工具，主要用于 MOD 管理、外置 HUD 显示、武器数据编辑以及本地插件集成。

## Features / 功能

- Mod management for Left 4 Dead 2
- External HUD overlays for speed, enemy health, and scoring data
- Weapon value editing and reference display
- Local plugin installation and parameter configuration
- Custom scoring rules for the Danyria scoring system
- Multi-language UI support
- Build scripts for Windows packaging

## 功能

- 《求生之路2》MOD 管理
- 速度、敌人血量、评分数据等外置 HUD
- 武器数值编辑与参考显示
- 本地插件安装与参数配置
- Danyria 评分系统自定义规则
- 多语言界面支持
- Windows 打包构建脚本
